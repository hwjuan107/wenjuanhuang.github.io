Convolutional Autoencoders¶

In this assignment, I used Convolutional Autoencoders to detect anomalies in images of Flowers Recognitionn dataset. This dataset contains 4,242 images of five categories: daisy, dandelion, rose, sunflower and tulip.

The specific tasks are:

  1. Created a Convolutional Autoencoders to detect anomalies of the images.
  2. Trained Autoencoder on the sunflower images. Output the same examples of the original vs. reconstructed images.
  3. Evaluated the Autoencoder to each one of the other flower types to detect anomalies and find the MSE between the normal and anomalous images.
  4. Output the sidtribution of density scores among training, validation, and anomalies for each flower type.
  5. Explained the how the autoencoder work for anomaly detection.
  6. Expalined what other methods can be used instead of anomaly detection.

1). Created a Convolutional Autoencoders

We treated images of sunflower as normal and after training the system using solely images of sunflower.

# Define the image size
IMG_SIZE = 96

# Load and preprocess the images
def preprocess_images(DIRECTORY):
    data = []
    for img in os.listdir(DIRECTORY):
        img_array = cv2.imread(os.path.join(DIRECTORY, img))
        img_array = cv2.cvtColor(img_array, cv2.COLOR_BGR2RGB)
        img_array = cv2.resize(img_array, (IMG_SIZE, IMG_SIZE))
        img_array = img_array.astype('float32') / 255.
        data.append(img_array)
    return np.array(data)

# Create a function for classifying images in bulk (for testing purposes)
def get_mse(original, reconstruction):
    # Returns the mean square error for each image in the array
    return np.mean((original - reconstruction)**2, axis=(1,2,3))

def evaluate_model(images,flower_type):
    density_threshold = 0 # This threshold was chosen based on looking at the distribution of the density scores of the normal class (validation set)
    reconstruction_error_threshold = 0.04  # This threshold was chosen based on looking at the distribution of reconstruction errors of the normal class

    reconstructions = sunflower_model.predict(images)
    MSEs = get_mse(images, reconstructions)
    reconstruction_anomalies = MSEs > reconstruction_error_threshold
    print(f'Reconstruction anomalies = {sum(reconstruction_anomalies)}')
    print(f'MSE between the normal and anomalous images {flower_type} = {mean(MSEs)}' )

    encoded_images = encoder_replica.predict(images)
    encoded_images_flat = np.array([np.reshape(img, (27)) for img in encoded_images])
    kde = KernelDensity(kernel='gaussian', bandwidth=0.2).fit(encoded_images_flat)
    density_scores = kde.score_samples(encoded_images_flat)
    density_anomalies = density_scores < density_threshold

    combined_anomalies = np.maximum(reconstruction_anomalies, density_anomalies)
    print(f'Combined anomalies = {sum(combined_anomalies)}')
    overall_detection_rate = sum(combined_anomalies) / len(combined_anomalies)
    print(f'Overall detection rate = {overall_detection_rate}')

5). Explained the how the autoencoder work for anomaly detection.

Autoencoder is a neural network that is trained on a dataset of normal data points, learns a compressed representation of the input data, and then reconstructs it back from the compressed epresentation.

During the training phase, the network minimizes the reconstruction error between the input and output data points. Once the autoencoder is trained, it can be used for anomaly detection by computing the reconstruction error between the original input and the reconstructed output.

Data points with high reconstruction error are likely to be anomalous.

Autoencoder-based anomaly detection is useful in situations where labeled training data is not available and can be used to detect anomalies in high-dimensional data.

6). Expalined what other methods can be used instead of anomaly detection.

Other methods for anomaly detection include statistical methods, clustering methods, Support Vector Machines (SVM), Isolation Forest, and Deep Learning methods such as Convolutional Neural Networks (CNNs) and Recurrent Neural Networks (RNNs).

Statistical methods use statistical properties of the data to identify anomalous data points.

Clustering methods identify anomalous data points that do not belong to any of the clusters or belong to small or sparse clusters.

SVM is a supervised machine learning method that can be used for anomaly detection.

Isolation Forest is an unsupervised machine learning method that randomly partitions the data into subsets to isolate anomalous data points.

Deep Learning methods can learn the patterns and features in the data to identify anomalous data points.

The choice of method depends on the data type, availability of labeled data, and required accuracy and efficiency of the anomaly detection system.
